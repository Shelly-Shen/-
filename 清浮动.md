#清浮动
###clear:both
缺点：增加了空白标签  
###overflow:hidden/auto
超出父元素的部分会被隐藏以及不能和position配合使用
###伪标签(伪元素是为#left内部最后加入元素)
	#left:after{
		content:"";
		display:block;
		height:0;
		lineheight:0px;
		visible:hidden;
		clear:both;
	}
	#left{
		zoom:1;
	}
###双伪标签
	#left:before,#left:after{
		content:"";
		display:block;
		clear:both;
	}
	#left{
		zoom:1;
	}
###给父元素加高度
